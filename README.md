# CarGreeting

CarGreeting is an Android app that automatically plays your greeting MP3 when your phone connects to Android Auto.

## What this app does

- Monitors Android Auto connection status
- Automatically plays a greeting when Android Auto connects
- Lets you choose a custom MP3 greeting
- Includes a **Preview greeting** button to test current audio

## How to use

### 1) Initial setup (important)

For reliable behavior and correct greeting playback:

1. Open Android Auto settings  
2. Turn off **“Automatically resume media”** / **“Auto-play music”**
3. Open CarGreeting
4. Tap **Enable Android Auto monitoring**

Recommended for background stability:

- Set battery mode for this app to **Unrestricted**
- Enable **Autostart / Startup** for this app (if your phone provides it)

---

### 2) Choose a greeting MP3

1. In the app, tap **Choose greeting MP3**
2. Select your MP3 file
3. Tap **Preview greeting** to test it

If you want to switch back, tap **Use default greeting**.

---

### 3) Daily use

1. Open the app once to make sure monitoring is enabled
2. Connect to Android Auto
3. The app will play your greeting automatically after connection

## Notes

- If no custom file is selected, the app uses the default `hello.mp3`.
- If greeting does not play:
  - make sure Android Auto **auto-play music** is turned off
  - increase media volume on phone/head unit
  - use **Preview greeting** to verify the current audio file
